# ziteh.github.io

This branch "source" was stored the source files of my blog.

## Basic HEXO Usage

- `hexo clean`
- `hexo g`
- `hexo d`
- `hexo s`

## Backup

```cmd
git remote add origin https://github.com/ziteh/ziteh.github.io.git
```

```cmd
git checkout source
git add .
git commit -m "Backup source files"
git push origin source
```
