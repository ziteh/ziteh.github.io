# ziteh.github.io

This branch "backup" was stored the source files of my blog.

## Basic HEXO Usage

- `hexo clean`
- `hexo g`
- `hexo d`
- `hexo s`

## Backup

```cmd
git remote add origin https://github.com/ziteh/ziteh.github.io.git
```

```cmd
git checkout -b backup
git add .
git commit -m "Backup"
git push origin backup
```
