KiCanvas——線上預覽KiCAD電路圖

2023-04-25

KiCanvas 是一個基於瀏覽器的開源 KiCAD Schematic、PCB 電路板檢視器。可以透過直接貼上 GitHub 連結或開啓本機檔案來查看 KiCAD 電路圖。

以往如果我想要查看 GitHub 上的 KiCAD 電路圖，都要把整個 repo 載下來，再用 KiCAD 開啓。但用 KiCanvas 就可以直接貼上網址，不用特別下載下來，對我來說還滿方便的。
